# 🌐 Recanto do Sagrado Coração (RSC)

> Projeto desenvolvido como atividade de extensão da disciplina de Construção de Páginas para Internet da Universidade Christus.

---

# ❤️ Sobre o Projeto

O objetivo deste projeto foi desenvolver uma página institucional acolhedora e informativa para o **Recanto do Sagrado Coração (RSC)**, uma casa de repouso para idosas.

A proposta faz parte de uma atividade de extensão universitária, aproximando os alunos da comunidade através de um projeto real, com foco em impacto social, acessibilidade e experiência do usuário.

A página foi criada pensando em:

- 📢 Divulgar o trabalho realizado pelo RSC
- 🤝 Incentivar doações e voluntariado
- 📍 Facilitar o acesso às informações da instituição
- 💻 Aplicar conceitos de desenvolvimento web utilizando HTML e CSS

---

# ✨ Funcionalidades

## 🏠 Cabeçalho Institucional

- Logo centralizada do RSC
- Nome da instituição destacado
- Layout acolhedor e responsivo

---

## 📑 Menu de Navegação

Menu interativo contendo links para:

- Sobre Nós
- Destaques
- Doações
- Localização

✨ Ao clicar nos links, a página realiza uma rolagem suave até a seção correspondente.

---

## 🎥 Seção "Sobre Nós"

- Vídeo institucional centralizado
- Conteúdo voltado à apresentação do Recanto do Sagrado Coração
- Estrutura organizada utilizando HTML semântico

---

## 🌟 Seção "Destaques"

- Imagem principal com bordas arredondadas
- Texto institucional justificado
- Informações sobre a missão e importância do RSC

---

## 🤝 Seção de Doações

Formulário completo para contribuição contendo:

- 👤 Nome completo
- 📧 E-mail
- 📱 Telefone
- 🎁 Categoria da doação
- 📅 Data da entrega
- 📝 Mensagem adicional
- 🙋 Checkbox para voluntariado

Além disso:

- Método `POST`
- Integração com envio via formulário
- Interface amigável e intuitiva

---

## 📍 Localização

- Integração com Google Maps via iframe
- Fácil visualização da localização da instituição

---

## 🔝 Rodapé

- Nome do desenvolvedor
- Botão/link para voltar ao topo da página
- Estilo moderno com fundo escuro
